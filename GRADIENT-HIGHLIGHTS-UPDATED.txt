NEW HIGHLIGHT SYSTEM:
=====================

✅ Gradient backgrounds (fade from color to transparent)
✅ HTML class toggle (tl-hl-on) controls all highlights
✅ Cleaner CSS with :not() selector for off state
✅ Individual toggles control visibility per type

CLASS MAPPING:
==============
AI-like      → .tl-hl-yellow (yellow gradient)
Polarization → .tl-hl-green  (green gradient)
Bias         → .tl-hl-red    (red gradient)

HOW IT WORKS:
=============

1. Highlights are applied with classes (tl-hl-yellow, etc.)
2. <html> gets "tl-hl-on" class when highlights are active
3. CSS shows gradients only when .tl-hl-on is present
4. Toggles show/hide individual highlight types
5. When all toggles off, "tl-hl-on" is removed
